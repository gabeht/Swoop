# Swoop
## IOS Rideshare Application
> **Note:** This project is currently in development and has not been fully released.

### Overview
Previously named "Guuber" - renamed to "Swoop" on February 6, 2026 for easier readability. This is an iOS application that connects users with appointed drivers for rideshare services.

### Technology Stack
- **Frontend:** iOS (Swift)
- **Backend:** Firebase
- **Additional Services:** MapKit

### Key Features
- Connect users to appointed drivers
- Real-time price monitoring for competitor services
- Price comparison display
